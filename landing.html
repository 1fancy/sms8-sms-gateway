  • Copy your API key from Profile → API.
  • Add the MCP server to your AI tool's config (below) and start coding.

MCP config

{
  "mcpServers": {
    "sms8": {
      "url": "https://mcp.sms8.io",
      "transport": "http",
      "headers": {
        "Authorization": "Bearer YOUR_SMS8_API_KEY"
      }
    }
  }
}
